# fliflc2pngs

Минималистичный конвертер FLI/FLC → PNG sequence

Небольшая автономная утилита для преобразования анимаций Autodesk Animator FLI/FLC в последовательность PNG-кадров.  
Размер бинарников: x64 ~47 KB • x86 ~39 KB

## Назначение
Конвертация файлов .fli и .flc в папку с пронумерованными PNG-кадрами. Без установки, без зависимостей, полностью офлайн.

## Возможности
- FLI / FLC → PNG sequence  
- Один исполняемый файл  
- Без библиотек и runtime  
- Быстрый запуск  
- Подходит для пакетной обработки  

## Использование
fliflc2pngs <input.fli|flc> <output_folder>

## Лицензия
Распространяется в виде бинарных файлов. Все права защищены.
